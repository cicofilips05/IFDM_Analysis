Análise do IFDM – Emprego e Renda (2013–2023)

Este projeto analisa o Índice Firjan de Desenvolvimento Municipal (IFDM) na dimensão Emprego e Renda entre 2013 e 2023, com estatísticas descritivas, rankings dos melhores e piores municípios, evolução temporal, médias por estado e mapas geoespaciais.

Conteúdo do Projeto

AnaliseIFDM.Rmd – Código completo do relatório em R Markdown.

AnaliseIFDM.html – Relatório gerado em HTML.

Dados/ – Base de dados utilizadas para a análise (ex.: IFDM Emprego & Renda).

Visualização

O relatório em HTML pode ser aberto diretamente em qualquer navegador, permitindo interatividade com gráficos e tabelas.

Como Executar

Instale os pacotes do R listados no AnaliseIFDM.Rmd.

Abra o arquivo AnaliseIFDM.Rmd no RStudio.

Clique em Knit para gerar o relatório em HTML ou PDF.
